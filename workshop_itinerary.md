# Intro to AI Engineering with Claude API

A live, one-day workshop that takes you from your first API call to building
tool-using, context-aware AI applications with Anthropic's Claude.

By the end of the day you will understand how to call Claude programmatically,
steer and evaluate its responses, ground it in your own data with RAG, extend it
with tools, and connect it to external systems through the Model Context Protocol
(MCP).

> **Who this is for:** Developers and technical professionals who want a practical
> introduction to building with large language models. Basic programming
> experience is helpful for following along, but no prior AI/ML background is
> required.

> **Format — mob coding:** This is a collaborative, presenter-driven session. We
> build everything together on one shared screen: I drive the code while the whole
> group participates — discussing the approach, designing prompts, and making
> decisions as we go. You learn by thinking through every step as a team, not by
> coding in parallel on your own machine.
